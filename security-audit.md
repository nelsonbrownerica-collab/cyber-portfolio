# Security Audit

This project demonstrates my ability to evaluate security controls, identify gaps, and align an organization with industry frameworks such as NIST CSF, SOC 2, PCI DSS, and HIPAA.

## Objectives
- Assess existing security controls
- Identify compliance gaps
- Recommend remediation steps
- Strengthen organizational security posture

## Scope
- Access control
- Network security
- Logging & monitoring
- Data protection
- Incident response readiness

## Summary of Findings
- Several controls partially implemented
- Logging inconsistent across systems
- No centralized SIEM correlation
- Access reviews not performed regularly

## Recommendations
- Implement centralized log management
- Enforce MFA across all systems
- Conduct quarterly access reviews
- Align policies with NIST CSF categories
